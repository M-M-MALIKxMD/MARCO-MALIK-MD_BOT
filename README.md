## 📦 Installation

### Prerequisites
- **Node.js 18+** — [Download here](https://nodejs.org)
- **npm** (comes with Node.js)
- A **WhatsApp account**

### Local Setup

```bash
# 1. Clone / Extract this project
cd marco-malik-md

# 2. Install dependencies
npm install

# 3. Copy and configure environment
cp .env.example .env
nano .env   # or use any text editor

# 4. Start the bot
npm start
```

📱 **Scan the QR code** that appears in your terminal with WhatsApp.

## 🚀 Deploy Guide

### 🚂 Railway (Recommended — Free)

1. Push this project to **GitHub**
2. Go to [railway.app](https://railway.app) → Login
3. Click **New Project → Deploy from GitHub Repo**
4. Select your repository
5. Add all **Environment Variables** (see below)
6. Click **Deploy** ✅

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template)

---

### 🎨 Render (Free)

1. Push to **GitHub**
2. Go to [render.com](https://render.com) → Login
3. Click **New → Web Service** → Connect repo
4. Set:
   - **Build Command:** `npm install`
   - **Start Command:** `node index.js`
5. Add Environment Variables
6. Click **Create Web Service** ✅

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

---

### 🟣 Heroku

1. Push to **GitHub**
2. Go to [heroku.com](https://heroku.com) → Login
3. Create new app → Connect GitHub repo
4. Add **Config Vars** (Environment Variables)
5. Enable **Automatic Deploys** → Deploy ✅

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

## ⚙️ Environment Variables

> Edit your `.env` file or add these as Config Vars in your deployment platform.

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `OWNER_NUMBER` | ✅ | — | Your WhatsApp number (country code, no +) |
| `OWNER_NAME` | ✅ | — | Your display name |
| `BOT_NAME` | ❌ | MARCO MALIK MD | Bot display name |
| `PREFIX` | ❌ | `.` | Command prefix |
| `BOT_MODE` | ❌ | public | `public` or `private` |
| `SESSION_ID` | ✅ | — | Your session ID |
| `GEMINI_API_KEY` | ❌ | — | Google Gemini API key |
| `OPENAI_API_KEY` | ❌ | — | OpenAI GPT API key |
| `REMOVEBG_API_KEY` | ❌ | — | Remove.bg API key |
| `AUTO_REACT` | ❌ | true | Enable auto react |
| `REACT_MODE` | ❌ | random | `random` or `fixed` |
| `FIXED_EMOJI` | ❌ | ❤️ | Emoji for fixed mode |
| `AUTO_READ` | ❌ | true | Auto read messages |
| `AUTO_STATUS_VIEW` | ❌ | true | Auto view statuses |
| `AUTO_STATUS_REPLY` | ❌ | true | Auto reply to statuses |
| `STATUS_REPLY_MSG` | ❌ | Seen Your Status 👀 By Marco Malik | Status reply text |
| `ANTI_CALL` | ❌ | true | Auto-reject calls |
| `ANTI_DELETE` | ❌ | true | Anti-delete messages |
| `VIEW_ONCE` | ❌ | true | View-once reader |

## 📱 Getting Session ID

**Method 1: QR Code** (Local only)
1. Run `npm start`
2. Scan the QR code with WhatsApp

**Method 2: Session Pairing** (For cloud hosting)
1. Visit a Baileys pairing website
2. Enter your WhatsApp number
3. Enter the 8-digit pairing code shown in WhatsApp
4. Copy the `SESSION_ID` string
5. Add it to your `.env` as `SESSION_ID=your_session_here`
