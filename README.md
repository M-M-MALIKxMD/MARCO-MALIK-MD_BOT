## 📦 DEPLOYMENT GUIDE

</div>

<details>
<summary><b>🚂 Deploy on Railway</b></summary>
<br/>

1. Click the **RAILWAY DEPLOY** button above
2. Sign in with your GitHub account
3. Fill in the required environment variables
4. Click **Deploy** and wait for the build to complete
5. Your bot is live! 🎉

</details>

<details>
<summary><b>🟣 Deploy on Heroku</b></summary>
<br/>

1. Click the **HEROKU DEPLOY** button above
2. Log in or create a Heroku account
3. Give your app a name
4. Set environment variables in the Config Vars section
5. Click **Deploy App** — you're done! 🚀

</details>

<details>
<summary><b>🟢 Deploy on Render</b></summary>
<br/>

1. Click the **RENDER DEPLOY** button above
2. Connect your GitHub account to Render
3. Select the repository to deploy
4. Add required environment variables
5. Hit **Create Web Service** and go live! ✅

</details>

<details>
<summary><b>🔗 Pairing Your Session</b></summary>
<br/>

1. Click the **PAIRING WEBSITE** button above
2. Enter your WhatsApp number
3. Scan the QR code or use the pairing code
4. Copy the session credentials
5. Paste into your deployment environment variables

</details>
